复刻天气时钟

kd_132024-06-072024-06-10

1. 复刻流程

嘉立创打板 => 购买元件 => 打印3D外壳 => 到货焊接 => 程序烧录 => 上电测试 => 组装

2. 打板

进入嘉立创开源广场搜 “天气时钟” ，找到图中的项目

也可以找到这个大佬的主页，他其中的这个时钟经过三个版本的迭代，上图为最终版本。

这里以最初的V1版本举例，其中三个三个版本的固件是同一个，只有板子的排列有所不同，都是一样的操作方式，进入V1版本的页面往下寻找板子资源图，在编辑器打开。

选择输出板子pcb文件后可以快速到下单页面

下面如果是免费的打样券都选择不需要即可。接下来下单等待即可，一般提交后厂家接单后开始生产需要2天完工，不同板子的颜色不同时间也不同，快递时效2天左右，大概下单5天左右就能拿到。

3. 原件购买

在板子下面就是需要的原件清单，电阻电容的最小售卖单位一般为100，所有按需购买。其中划掉的可以不焊接

这里提供的BOM清单缺少0.1uF电容，以上原件在3代产品中，除了MICRO USB转接板只有在1代中使用，2、3代都直接焊接在板子上，可以直接使用2、3代更加简洁的板子进行焊接

4. 焊接

贴片电容焊接需要一点手法，可以用电烙铁也可用加热台配合锡膏，看个人能力选择，焊接位置可以参考BOM表

也可以在刚才的编辑器中选择3D预览，特别注意那个电容不能焊接为电阻，否则无法烧录程序

这里的ESP-12F原件位置错误了，U2触摸也可以不用焊接

5. 程序烧录

图中的UART4个口子就是上传烧录使用的，这里使用ch340烧录

这样就可以直接插上烧录，一定要按住BOOT按键再插上去，一直按着直到出现连接时才可以松手

5.1 二进制烧录

这是最简单、最快捷的烧录方式，下载

在这个齿轮的位置打开下载好的bin文件，选择你的ch340端口号点击烧录，如果出现很多，可以拔下再插上看看哪个端口消失了。

5.2 Arduino烧录

第一次安装的话需要在版本中安装ESP8266的开发板，具体办法百度或者留言，如果出现库不存在的问题，百度库的安装或者将文件中的库移动的指定位置。

每次启动软件，点击上传都会重新编译二进制文件，第一次需要个半分钟左右，建议先编译在点击上传分开操作，不然手按着BOOT按钮很痛，下图则编译成功

这时点击上传，按住BOOT按键插上电等待下图….____显示连接就可以松手。

松手后出现下图则表示开始烧录

下图表示烧录成功

记得连date文件一起烧录，可以看嘉立创项目里面的介绍

6. 组装

插上屏幕焊接号电源线后安装手册进行配置就可以使用。

7. 外壳

外壳还在快递中。。。。
